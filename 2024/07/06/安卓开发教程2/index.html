安卓开发教程2

本期接着上期教程，教学如何配置安卓手机环境，设置软件图标，安装自己的软件到手机上

准备材料：1、安卓手机 2、电脑 3、数据线

首先打开安卓手机设置，我的设备是小米10s打开设置找到关于手机，点击全部参数，点击系统版本，每个手机的打开方式不一样，网上自行搜索自己的手机如何打开开发者选项，打开开发者选项后，我们到设置中看到更多设置下滑即可看到开发者选项，我们点击进入下滑找到USB调试，打开USB安装，通过数据线连接后，手机会弹出窗口，是否授权该电脑对手机的调试，我们选择是即可，授权成功后打开android studio我们之前的项目，如果前面的USB配置没有问题那么你将会看到以下东西

如果成功到了这个位置那么接下来我们就开始教学配置安卓app的图标了

在上一期中我们有讲过res是资源目录，所以我们标准开发也需要去这个目录中添加你想要的图标，首先我们需要准备一个正方形的图像，因为app图片基本都是正方形，建议用矢量图，如果普通图标可能会导致有显示不清晰的问题，准备好了后我们找到res目录mipmao这个目录，如图：

点开后如图

Icon type 是图标类型，我们默认即可 Name 是图标名称，我们后期是需要去调用所以这个得你记住(不能是中文)在安卓开发只要是代码或者调用部分都不能用中文，所以这里我们随便娶一个 然后就是Path这个是图标的路径，如果你想要修改自己喜欢的图标就指出图片的路径和图片名称，选择好了后就可以看到预览图片了 Icon type这个是调整图像的大小和参数的，最终在安卓上呈现的图像是最后俩个一个是正方形一个是圆形，这个是为了适配不同手机的不同尺寸当一切设置完成后大概效果如图

然后我们点击下一步，会出现很多文件，这些文件都是适配不同设备而生成的，我们在确定信息无误后点击确定即可，然后我们就可以在mipmao目录中看到有三个你刚才命名的文件

接下来我们讲解修改代码，还记得我们上一章讲过的吗？在app/manifests/目录下的AndroidManifest文件吗？没错就是安卓权限目录，我们点击进入，会发现代码旁边有2个安卓图标的东西，没错这个就是要显示的图标我们需要把他改成我们需要的图标，如图

这样就可以了，我们点击运行，因为我们是需要看效果的所以就直接点击运行，而不是编译，编译是为了方便观察打包是否会报错，运行是指要在安卓上使用，我们是需要看到效果的

点击运行后稍等片刻，我们就可以看到安卓手机提示app安装了，我们点击继续安装后，属于你自己的app就正式诞生了
